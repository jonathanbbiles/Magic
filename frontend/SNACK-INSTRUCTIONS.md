# Snack Instructions

1. In Snack, replace the contents of `App.js` with the contents of `App.snack.js`.
2. Create a new file named `StageDashboardScreen.js` in Snack and paste the contents of `StageDashboardScreen.snack.js`.
3. Update the import in Snack `App.js` to:

   ```js
   import StageDashboardScreen from "./StageDashboardScreen";
   ```
