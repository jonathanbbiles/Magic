# Magic

## Repository Layout

- `backend/`: existing backend services, routes, models, and business logic.
- `frontend/`: rebuilt Expo frontend (“Magic Mission Control”).
- `shared/`: shared utility modules used by backend.

## Frontend Rebuild Policy

The frontend is intentionally decoupled from backend internals. Backend logic was not modified during this rebuild.

For frontend architecture, run/setup, guardrails, and full tree documentation, see `frontend/README.md`.
